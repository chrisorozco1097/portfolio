Safety Scoreboard Tracker

Digitized safety incident tracking system with automated reporting.

Technologies

Power BI
PowerApps
MySQL
DAX
SharePoint
Power Automate

Objective

Previously, the Safety Department relied on a whiteboard to manually track workplace incidents — a process that was error-prone, time-consuming, and lacked historical insights.
My goal was to digitize and automate this process, enabling real-time tracking, trend analysis, and data-driven decision-making.

My Role & Techniques

1. Custom Incident Submission App

  • Developed a PowerApps form tailored for the Safety Director to log incidents with structured fields (e.g., Shop, Incident Type, Body Part Affected) ensuring consistent data collection.
Incident submission via PowerApps.

2. Data Storage with SharePoint

  • Configured a SharePoint List as a central repository for incident submissions, allowing cross-platform access and data integrity.

3. Automated ETL with Power Automate

  • Built a Power Automate Flow to:
    • Extract new incident data daily
    • Organize incidents by month
    • Calculate department-defined KPIs
    • Clean and structure data for reporting

4. Interactive Power BI Dashboard

  • Designed a Power BI report displaying:
    • Monthly trends and heatmaps
    • KPIs segmented by Shop, Body Part, and Date
    • Active-employee filtering through HR data integration
    • Filters and drill-through features for root cause analysis
Monthly trends in Power BI and KPI analysis.

Results & Impact

  • Replaced manual tracking with a fully automated workflow
  • Reduced update time from hours per week to minutes per day
  • Enabled real-time visibility into safety trends and compliance metrics
  • Empowered management to identify and mitigate high-risk areas faster

Why It Matters

Safety is a foundational element in any organization, and this project transformed safety management from a reactive, manual process into a proactive, data-driven system.
By digitizing the incident reporting workflow, the Safety Department now has instant visibility into safety trends, allowing early detection of recurring issues and enabling faster interventions.

The integration of automated data flows and interactive dashboards empowers leadership to focus on prevention rather than paperwork, ensuring that safety concerns are addressed in real time.
This solution also promotes organizational accountability by standardizing data collection and providing transparent reporting that can be accessed at any time.

Ultimately, the Safety Scoreboard helps reduce workplace risks, improve employee well-being, and supports the creation of a safer, more responsive work environment while also significantly saving administrative time.
