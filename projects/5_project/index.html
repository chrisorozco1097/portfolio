Invoice Automation - Energy Management

Automated invoice reading and processing using AI and data pipelines to streamline utility billing in AiM.

Technologies

SharePoint
Power Automate

Objective

Manually processing over 350 monthly utility invoices was a time-consuming and error-prone process for the Energy Management team. Each invoice had to be reviewed, entered line by line into AiM, and validated, often taking multiple days per month.

The goal of this project was to create a fully automated pipeline to extract, process, and upload invoice data using AI and automated workflows, reducing manual work to a few minutes per month while improving accuracy and reporting speed.

My Role & Techniques

1. AI-Powered Data Extraction

  • Created a custom AI Builder model to intelligently read utility invoices in PDF format.
  • The model was trained on historical invoices from different vendors, extracting key fields such as account numbers, billing periods, amounts, and line item details.

2. Power Automate Invoice Processing Flow

  • Built an end-to-end Power Automate flow to:
    • Monitor the receipt of invoices in a designated SharePoint folder
    • Automatically extract the files and send them to AI Builder for processing
    • Format the AI-extracted data into structured records and validate common errors
    • Aggregate all invoices into a standardized JSON file for bulk loading

3. JSON Data Consolidation

  • Developed logic to combine all extracted invoices into a single JSON file containing over 350 records per month, each aligned with the required structure for AiM’s energy management module.

4. Seamless AiM Integration

  • Enabled a single-point upload into AiM, allowing the Energy Management team to import all invoices at once without the need for manual entry.
  • Added tracking fields for quick validation post-import.
Automated workflow from PDF ingestion to bulk AiM upload.

Results & Impact

  • Reduced invoice processing time from several days to less than 15 minutes per month
  • Fully eliminated manual data entry for over 350 monthly invoices
  • Standardized data formatting improved auditability and consistency
  • Established a scalable model for future invoice types and vendors
  • Allowed Energy Management to focus on analysis and strategy, not data entry

Why It Matters

Utility billing is a critical operational process directly tied to financial accountability and energy usage optimization.
Manual invoice entry not only consumed valuable resources but also introduced risks of human error, delayed reporting, and inconsistencies in billing data.

This project transformed the process into a streamlined, hands-free workflow.
Leadership now benefits from timely, accurate data, quicker month-end closings, and the flexibility to scale the process across multiple utilities and account types.

It demonstrates the effective combination of AI capabilities and low-code automation to solve real administrative bottlenecks while providing a foundation for future operational improvements.
